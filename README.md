# STUDENT PREDICTION AND DATA MANAGEMENT SYSTEM
**A Robust Windows Forms Application for Academic Performance Analysis**

---

## 1. PROJECT INTRODUCTION
Student Prediction System ek specialized desktop application hai jo C# aur .NET Framework ki madad se develop ki gayi hai. Is software ka buniyadi maqsad educational records ko digitally maintain karna aur student ke marks ki buniyad par unki academic success ka ta'yun (prediction) karna hai. 

Application ko simple rakhte hue professional standards ko mad-e-nazar rakha gaya hai taky data entry aur result analysis mein ghalti ka imkan kam se kam ho.

---

## 2. KEY ARCHITECTURAL FEATURES

### A. Data Input Module
User interface mein char mukhtalif labels aur textboxes faraham kiye gaye hain:
* **Student Name:** Full identity capture karne ke liye.
* **Age:** Student ki age verify karne ke liye.
* **Student ID (Reg No):** Har student ko unique identity dene ke liye.
* **Academic Marks:** Result calculate karne ke liye numerical data input.

### B. Logical Processing (Prediction Engine)
System ka core logic marks ko analyze karta hai. Jab "Display Result" button trigger hota hai, toh system logic check karta hai:
* **Pass Status:** Agar marks required passing criteria se upar hon.
* **Fail Status:** Agar marks criteria se kam hon.

---

## 3. FUNCTIONAL BUTTONS DESCRIPTION

- **Add Student:** Ye button user ke input kiye hue data ko collect karke memory mein save karta hai.
- **Display Result:** Ye logic run karta hai aur batata hai ke student pass hai ya fail.
- **Show All Records:** Ye button ek naya view faraham karta hai jismein tamam save kiye gaye students ka data show hota hai.
